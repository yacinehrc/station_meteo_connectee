# station_meteo_connectee


~ PRÉSENTATION ~

Ce projet MSPR (Mise en Situation Profesionnelle Reconstituée) a pour objectif de nous mettre en situation afin de créer une station-météo connectée à l'aide d'une Arduino Uno et d'un capteur DHT 11 (qui seront finalement 
accompagnés d'une breadbord, de câbles Dupont et d'un écran LCD), et aussi de découvrir le monde du 3D (modélisation, éco-conception, géométrie dans l'espace, ...). Nous avons donc utilisé le langage Arduino qui fût une 
découverte pour nous.
